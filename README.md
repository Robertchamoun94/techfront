# TechFront Website

Färdig React + Vite hemsida för TechFront.

## Starta lokalt

```bash
npm install
npm run dev
```

## Build-testa före GitHub/Vercel

```bash
npm run build
```

## Deploy till Vercel

1. Pusha hela mappen till GitHub.
2. Importera repot i Vercel.
3. Vercel känner igen Vite via `vercel.json`.
